//React's goal is in many ways to render HTML in a web page. React renders HTML to the web page by using a function called ReactDOM.render().

// The Render Function
// The ReactDOM.render() function takes two arguments, HTML code and an HTML element.
// The purpose of the function is to display the specified HTML code inside the specified HTML element.

// Display a table inside the "root" element:
//Create a variable that contains HTML code and display it in the root node:
const myelement = (
  <table>
    <tr>
      <th>Name</th>
    </tr>
    <tr>
      <td>John</td>
    </tr>
    <tr>
      <td>Elsa</td>
    </tr>
  </table>
);

// The result is displayed in the <div id="root"> element in index.html:
ReactDOM.render(myelement, document.getElementById('root'));
// Name
// John
// Elsa

/* The Root Node
The root node is the HTML element where you want to display the result.
It is like a container for content managed by React.
It does NOT have to be a <div> element and it does NOT have to have the id='root':
